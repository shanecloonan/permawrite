          Cleared checkpoints resume via get_light_snapshot at the last scanned height (M4.12).
          Optional quorum RPC URLs, P2P peers, or ≥2 independent light relays (M4.14–M4.17);
          trusted checkpoint summaries, relay URL TOFU (M4.18), per-relay summary pins (M4.19),
          pasted weak-subjectivity import (M4.22), and export (M4.23) live in localStorage.
